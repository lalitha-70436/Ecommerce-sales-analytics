# 🚀 E-Commerce Sales Analytics Web Application

A full stack + data analytics web application that helps businesses manage, track, and analyze their sales data through a single interactive dashboard.

---

## 📌 Problem Statement

Small business owners often rely on Excel sheets and manual methods, which leads to:

- ❌ No centralized data management  
- ❌ No real-time sales insights  
- ❌ Manual and error-prone operations  
- ❌ No visual reports (charts/graphs)  

---

## 💡 Solution

This project provides a complete dashboard system that:

- Centralizes products, customers, and orders  
- Generates real-time analytics  
- Visualizes data using charts  
- Simplifies business decision-making  

---

## 🛠️ Tech Stack

### Frontend
- HTML5  
- CSS3  
- JavaScript  

### Backend
- Python (Flask)  
- REST APIs (Flask Blueprints)  

### Database
- MySQL (Normalized schema with foreign keys)  

### Analytics
- Pandas  
- Matplotlib  
- Seaborn  

### Authentication
- Werkzeug Password Hashing (secure password storage)  
- Flask Sessions (user login management)  

---

## 📊 Key Features

### 🔐 Authentication
- Role-based login (Admin & Customer)  
- Secure password hashing  
- Session-based login tracking  

### 📦 Management System
- Product Management (Add / Edit / Delete)  
- Customer Management  
- Order Management with status tracking  

### 📈 Analytics Dashboard
- Monthly Revenue Trends  
- Top 5 Selling Products  
- Category-wise Revenue Distribution  
- Order Status Analysis  

---

## 📁 Dataset

- Superstore Dataset  
- 9,800+ real-world records  

---

## 🧠 What I Learned

- Building REST APIs using Flask  
- Designing normalized MySQL databases  
- Connecting Python with MySQL  
- Data analysis using Pandas  
- Data visualization using Matplotlib & Seaborn  
- Full stack architecture  

---

## 🏗️ Project Architecture

- Frontend Layer → UI (HTML, CSS, JS)  
- Backend Layer → Flask APIs  
- Database Layer → MySQL  
- Analytics Layer → Pandas + Matplotlib
